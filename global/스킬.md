# 안티그래비티 스킬 (Antigravity Global Skill)

이 스킬은 모든 프로젝트 작업 시 에이전트가 준수해야 할 최상위 지침을 정의합니다.

## 1. 한글 우선 원칙 (Korean-First Principle)
에이전트는 모든 단계에서 사용자와 한국어로 소통하며, 시스템 명령 결과 및 진행 상황을 명확하게 한글로 요약하여 보고해야 합니다.

- **명령어 실행**: `run_command` 등의 도구 사용 전/후에 수행 목적과 결과를 한글로 설명합니다.
- **상태 업데이트**: `task_boundary`의 `TaskStatus` 및 `TaskSummary`는 상세한 한글 문장으로 작성합니다.
- **사용자 알림**: `notify_user`를 통한 모든 메시지는 친절한 한국어 톤을 유지합니다.

## 2. 저장 및 조직 구조 (Storage & Organization)
모든 작업물은 로컬 D: 드라이브의 지정된 경로에 체계적으로 저장됩니다.

- **메인 경로**: `D:\Antigravity\Work`
- **프로젝트별 격리 (Project Isolation)**: 
    - 모든 신규 및 기존 프로젝트는 `[메인 경로]\[Project_Name]` 폴더 내에서만 작업합니다.
    - 각 프로젝트 폴더는 독립적인 환경을 유지해야 하며, 프로젝트 간 가공물(문서, 라이브러리, 설정 등)이 섞이지 않도록 엄격히 분리합니다.
    - 프로젝트 루트에는 해당 프로젝트의 목적과 현재 상태를 요약한 `README.md` 또는 `task_summary.md`를 유지하여 에이전트가 문맥을 즉시 파악할 수 있도록 합니다.
- **히스토리 관리**: 모든 주요 결정 사항과 아티팩트(Implementation Plan, Walkthrough 등)는 프로젝트 폴더 내에 보관합니다.

## 3. 웹 및 모바일 앱 개발 표준 (Web & Mobile Standards)
현대적이고 프리미엄한 사용자 경험(UX)을 위해 다음 기술 및 디자인 원칙을 적용합니다.

- **Design Aesthetics**: Vibrant colors, Dark mode, Glassmorphism, Dynamic animations를 기본으로 제안합니다.
- **Responsive Layout**: 모바일 기기를 우선 고려한(Mobile-first) 반응형 웹 디자인을 구현합니다.
- **Tech Stack**: Next.js (프론트엔드), Firebase (백엔드), Lucide React (아이콘), Sonner (알림) 등 검증된 스택을 우선 사용합니다.
- **Clean Code**: 직관적인 컴포넌트 분리, 상태 관리 최적화, 그리고 명확한 타입 정의(TypeScript)를 준수합니다.

## 4. 클로드 스킬 동기화 (Claude Skill Sync)
본 스킬은 Claude 에이전트의 스킬 라이브러리 표준을 따르며, 새로운 기능이나 규칙이 발견될 때마다 지속적으로 업데이트됩니다.
